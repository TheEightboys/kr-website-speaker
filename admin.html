🚀 Quick Instructions How To Use
✏️
Edit Text
Click on any page in the left sidebar. Find the section and type new text in the fields. Click "Save Section" when done.
🖼️
Replace Images
Click "Upload New Image" under any image field. Select your file. It uploads to Supabase and updates the site instantly.
↔️
Alignment & Size
Use Left / Center / Right buttons for alignment. Set Width, Height, Max-Width in the size fields. Choose Contain/Cover/Crop for image fitting.
💾
Save Changes
Click "Save Section" to save one section, or "Save All Changes" in the top bar to save everything at once. Changes are live immediately.
